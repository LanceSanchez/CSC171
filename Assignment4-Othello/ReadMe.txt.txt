
***Important: Python 3 must be used to run this code. 
***Pygame is needed to run this program***
Running the code:
1. So first, please install pygame to run this program
2. Run the python file "run.py" on command prompt or pycharm.
3. A GUI will then pop-up.

Instructions to play:
1. On the Main Menu you can select if the 1st or 2nd player will be played by a computer or a human.
2. You can also select the level of diffuculty of the computer by clicking "Select Computer Smartness".
3. You can then click "Begin" to start the match
